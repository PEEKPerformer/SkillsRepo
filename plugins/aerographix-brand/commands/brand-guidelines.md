# Aerographix Brand Guidelines

Apply Aerographix visual identity to any artifact. Dark-mode native design conveying scientific precision and clean air technology.

## Quick Reference

### Colors
| Name | Hex | Use |
|------|-----|-----|
| Electric Blue | `#00A8E8` | Primary accent, links, CTAs |
| Plasma Cyan | `#00D9FF` | Highlights, gradients, glow |
| Carbon Black | `#0A0A0F` | Primary background |
| Graphite | `#1A1A24` | Cards, secondary bg |
| Steel | `#2D2D3A` | Borders, dividers |
| Silver | `#8B8B9E` | Body text |
| White | `#F5F5F7` | Headlines |

### Gradients
```css
--gradient-primary: linear-gradient(135deg, #00A8E8, #00D9FF);
--gradient-heat: linear-gradient(135deg, #FF6B35, #FF8C42); /* thermal mode */
```

### Typography
- **Font:** DM Sans (Google Fonts)
- **Headlines:** 700 weight, #F5F5F7, letter-spacing: -0.02em
- **Body:** 400 weight, #8B8B9E, line-height: 1.6
- **Labels:** 600 weight, uppercase, letter-spacing: 0.1-0.2em

### Key Elements
```css
/* Glow effect */
box-shadow: 0 0 30px rgba(0, 168, 232, 0.3);

/* Card */
background: linear-gradient(145deg, #1A1A24, rgba(26, 26, 36, 0.8));
border: 1px solid #2D2D3A;
border-radius: 16px;

/* Primary button */
background: linear-gradient(135deg, #00A8E8, #00D9FF);
color: #0A0A0F;
```
